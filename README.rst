GUILLOTINA_CMS
==============

WIP: This package is a work in progress to provide CMS on guillotina

Bundle of cms functionality for guillotina

Bootstrap dev
-------------

docker-compose start postgres

Compile Pastanaga
-----------------

cd pastanaga
yarn install
npm run build
cd ..

Compile Plone React
-------------------

cd plone-react
yarn install
npm run build
cd ..

Run dev
-------

docker-compose up guillotina
